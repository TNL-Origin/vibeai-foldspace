Summary

VibeAI FoldSpace processes all data locally in your browser. No external servers. No telemetry. No tracking.

What Data We Access

VibeAI FoldSpace is a browser extension that enhances your experience on AI chat platforms (ChatGPT, Claude, Gemini, Microsoft Copilot). To provide its functionality, the extension accesses:

  1. Visible Chat Content ‚Äî FoldSpace analyzes only on-screen messages in your active chat view to render emotional tone and code overlays
  2. Page Metadata ‚Äî Current URL and page title to detect which platform you're using
  3. DOM Content ‚Äî Visual elements on the page to inject the VibeAI overlay interface

Local Processing Only

All analysis occurs inside your browser. No message content is transmitted or stored externally. The extension reads visible messages purely for real-time, in-memory tone analysis and code detection.

No History Access

Past or hidden conversations are never accessed. Only messages currently visible on your screen are processed, and only while you're actively viewing that thread.

What Data We Store

All data storage is local to your browser:

Chrome Storage (Synced Across Your Chrome Profile)

  • Thread Analysis Toggle (enable/disable tone analysis)
  • Canvas Toggle (enable/disable FoldSpace visual overlay)
  • Beta Consent Status (whether you accepted the beta agreement)

Temporary In-Memory State

  • Tone Visualization Data ‚Äî Calculated tone scores exist only while messages are visible, cleared when you navigate away
  • No Persistent Message Storage ‚Äî Message content is never saved to disk or Chrome storage
